(Columns: ustomer_id and purchase_date)

ğŸ SQL code:

SELECT
customer_id,
DATEDIFF(MAX(purchase_date), MIN(purchase_date)) AS days_between_purchases
FROM
emp
GROUP BY customer_id;


ğŸˆ MAX(purchase_date) retrieves the latest purchase date for each customer.
ğŸˆ MIN(purchase_date) retrieves the earliest purchase date for each customer.
ğŸˆ DATEDIFF() calculates the difference in days between the latest and earliest purchase dates.
ğŸˆ GROUP BY customer_id groups the results by customer, 
so you get the number of days between the first and last purchase for each customer.
